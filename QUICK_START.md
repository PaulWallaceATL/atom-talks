# Quick Start Guide - AtomTalks

Get your AtomTalks site running in 2 minutes!

## Super Simple Setup (No Database!)

### Step 1: Install

```bash
git clone https://github.com/PaulWallaceATL/atom-talks.git
cd atom-talks
npm install
```

### Step 2: Run

```bash
npm run dev
```

### Step 3: Open Browser

- **Website:** http://localhost:3000
- **Contact:** http://localhost:3000/contact
